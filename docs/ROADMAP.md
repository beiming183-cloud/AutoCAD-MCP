# Roadmap

AutoCAD MCP prioritizes trustworthy automation over a long list of unverified commands.

## Now: 3.10

- Reliable document identity, optimistic revisions, and transactions.
- Checked 2D entity creation and topology-aware DRC.
- Native 3D primitives and analytic rounded product features.
- Fixed-camera native previews with deterministic framing evidence.
- Motion and clearance screening clearly labeled as broad-phase.

## Next

- Stable semantic face/edge selection through a native feature plugin.
- General 3D fillet, chamfer, shell, draft, loft, mirror, and pattern operations.
- STEP export/re-import comparison and B-rep validity reports.
- Component instances, configurations, mates, and exact interference checks.
- Reproducible multi-view drawing generation from one 3D authority.

## Later

- Hybrid OpenCascade/FreeCAD execution behind the same document and evidence contracts.
- Native section and exploded-view state generation.
- Wall-thickness, draft-angle, and manufacturability analysis.
- Signed release artifacts and package publication.

Contributions should include deterministic acceptance evidence and should never turn an
unavailable check into a favorable result.
