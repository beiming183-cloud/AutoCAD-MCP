# Code of Conduct

We want this project to be a practical, respectful place for CAD users, engineers,
developers, and first-time open-source contributors.

- Be respectful and specific when discussing defects or design tradeoffs.
- Critique code and evidence, not people.
- Welcome questions from users who are new to AutoCAD, Python, or MCP.
- Do not harass, threaten, discriminate against, or expose private information about others.
- Avoid promotional spam and fabricated benchmarks or capability claims.

Maintainers may edit or remove comments, issues, pull requests, or contributions that
violate these expectations. Serious or repeated violations may result in a ban.

Report conduct concerns privately through the maintainer's GitHub profile contact.
